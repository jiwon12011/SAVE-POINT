# SAVE POINT

> 지친 하루를 RPG 상태창으로 번역하고, 오늘 가능한 "최소 퀘스트"를 제안하며, NPC 조력자가 회복 대사를 건네는 서브컬처 루틴 앱.
>
> **"오늘의 나를 저장하고, 다시 시작할 힘을 얻는 곳."**

핵심 루프: **체크인 → 상태창 생성 → 퀘스트(난이도 조정) → NPC 대화 → 세이브 완료 → 기록 누적**
원칙: 퀘스트를 다 못 깨도 **세이브는 된다.** "아무것도 안 한 날"을 만들지 않는 것이 이 앱의 존재 이유.

## 확정 사항
- **형태:** 실제 모바일 앱 (Next.js + React + TS + Tailwind + Supabase + Capacitor)
- **AI:** 하이브리드(규칙 엔진 뼈대 + LLM 문구) · 제공자 미확정(Grok 유력) · P0는 템플릿, P1에서 LLM
- **데이터:** 계정 + 클라우드(Supabase) + 게스트 모드
- **디자인:** 귀여운 판타지(Cozy RPG) — 파스텔·치비·따뜻한 밤 · 다크 전용(MVP)
- **NPC:** MVP 2종(힐러·여관주인) → P2에서 3종 확장

> 현재 단계: **기획**. 구현은 아직 시작하지 않음.
