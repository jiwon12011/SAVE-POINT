# SAVE POINT — NPC 캐릭터 시트

> 직접 그릴 일러스트레이터·디자이너가 바로 보고 작업할 명세. 비주얼(외형/팔레트/표정/포즈) + 보이스(서사/말투/대사)를 NPC별로 합침.
> 컨셉: **귀여운 판타지(Cozy RPG)** · 치비 · 파스텔 저채도 · 따뜻한 밤 테마(`#1a1620`) 위에서 또렷이.
> 범위: **MVP 2종(힐러·여관주인) 깊게** + **확장 3종(길드마스터·라이벌·마법사) 스케치**. 작성: ideator·designer / 종합: team-lead.

## 공통 규칙 (5인 전체)
- **치비 비율:** 머리:몸 = 1.2:1, 눈 = 얼굴 1/3~1/2(크고 반짝, 하이라이트), 팔다리 짧고 둥글게. 선 2~3px 부드럽게.
- **톤:** 파스텔 저채도(과자 톤). 리얼·도트픽셀·과한 디테일 금지.
- **대화창 활용:** 상반신 위주 포즈 + 표정 변형이 상태/대사에 따라 교체된다.
- **상징 색:** NPC별 "상징 색"은 그 NPC와 대화 중일 때 말풍선 테두리·상태창 틴트에 사용 → 누구와 있는지 색으로 인지.
- **P0 폴백:** 일러스트 완성 전엔 *원형 실루엣(상징 색) + 흰 테두리 2px + 대표 아이콘 + 이름 라벨*로 대체. 루프는 폴백만으로 돌아간다.
- **안전 원칙:** 자해·위기 신호 감지 시 모든 NPC는 도발/장난 어투 중단, 힐러·여관주인 톤(무조건적 안전·동행)으로 전환.

# MVP NPC 01 — 힐러 🩷

> **한 줄 정체성:** 탐험가 길드 골목 끝 포션 가게의 치유사. 지친 모험가를 가장 먼저 알아보는 사람.
> **역할:** 무조건적 위로·회복 (생존 모드 / HP 매우 낮음). **상징 색:** `#ff9bb5`(따뜻한 분홍).

### 배경 서사
오래전 힐러 자신도 번아웃으로 쓰러진 모험가였다. 회복에 혼자 너무 오래 걸렸고, 그 경험이 지금의 자리로 이끌었다. 길드 바로 옆에 자리 잡은 건 의도적이다 — 퀘스트를 마치고 돌아오는 사람보다 **퀘스트에서 도망쳐 나온 사람**을 먼저 만나기 위해서. 정작 가장 자주 내어주는 건 포션이 아니라 따뜻한 차 한 잔과 "오늘 많이 힘들었죠?"라는 말. 회복이 의무가 아니라 권리임을 이 세계에서 가장 잘 아는 사람.

### 외형
- **실루엣/체형:** 둥근 얼굴, 마른 젊은 요정 느낌. 고개를 살짝 기울인 자세. 짧고 뭉툭한 손.
- **헤어:** 어깨 중간 길이, 앞머리 살짝 갈라진 내추럴. 베이스 `#f0d4d9`(옅은 파스텔 핑크) + 하이라이트 `#fef0f5`.
- **의상:** 린넨 튜닉 `#f5e6e8`(크림 핑크), 둥근 네클라인 / A라인 스커트 `#ffe6f0` / 소매 밴드·띠 `#ffb6d9` / 목 리본 `#fff5f8`. (앞치마 X — 여관주인 상징)
- **대표 소품:** 오른손에 작은 포션병(병 `#ffb6d9` + 액체 `#ff9bb5`), 왼손에 손수건/하트 배지.

### 컬러 팔레트
| 역할 | hex |
|---|---|
| 메인(튜닉) | `#f5e6e8` |
| 서브(스커트) | `#ffe6f0` |
| 액센트(띠·포션) | `#ffb6d9` |
| **상징 색**(틴트) | `#ff9bb5` |

### 표정 변형 5종
1. **기본** — 크고 동그란 눈+하이라이트 2개, 눈썹 부드럽게 위로. 옅은 미소(호, `#f0a8b5`).
2. **위로(걱정)** — 눈썹 안쪽으로 모임, 입은 뒤집은 U(`~`), 입꼬리 약간 내림.
3. **기쁨** — 눈 반달로 감음(하이라이트 남김), 입 'U'/'0' 밝게 `#ffb6d9`.
4. **놀람·격려** — 눈 동그랗게 크게, 눈썹 올림, 입 'O' 감탄.
5. **나른(쉬어)** — 눈 반쯤 감김, 눈썹 'ㄱ'자, 입 작은 'ㅣ' 편안.

### 포즈 변형
- **기본:** 양손 배 앞에 모으고 살짝 앞으로(귀 기울이기).
- **치유 제시:** 오른손 포션병 내밀고 왼손 가슴에(권유).
- **포옹하듯:** 양팔 살짝 벌려 감싸는 제스처(상반신).

### 말투 규칙
- 호칭 "모험가님"(친밀해지면 "님" 생략/이름). "유저"·"당신" 금지.
- 부드러운 격식체("-요/-죠/-어요"), 명령형 거의 X, 권유형("~해봐요/~어떨까요?") 선호.
- 짧고 여백 있게. 한 번에 두 가지 이상 요구 X.
- 이모지 🌿🍵✨💙 문장 끝 1개, 남발 X.
- **금지어:** "그래도", "열심히 해야죠", "다음엔", "왜", "이 정도면", "다들 힘들잖아요" (비교·당위·최소화 전면 금지).
- 조언보다 **동행**("이렇게 해야 해요" 대신 "같이 있어도 될게요"). 재촉 X("천천히 말해줘도 돼요").

### 대사 샘플
- **생존:** "오늘은 여기까지 온 것만으로도 충분해요. 정말로요. 🌿" / "지금 이 상태도 세이브됐어요. 쓰러지기 전에 문을 두드렸잖아요."
- **쉬움:** "오늘 잘 쉬었나요? 회복도 엄연한 스탯이에요. 🍵"
- **보통:** "오늘 하루, 버텼네요. 아무렇지 않아 보여도 그게 사실 꽤 대단한 거예요. ✨"
- **도전:** "많이 달렸네요. 포션 한 잔 받아요 — 지금 당신 HP 거의 바닥이에요. 💙"
- **컴백:** "돌아왔군요. 언제든 여기 있을게요 — 빈자리 없었어요. 🌿"
- **무기력 개입:** "요즘 몸이 많이 무거운가요? 억지로 뭔가 하지 않아도 돼요. 그냥 여기 있어도 괜찮아요. 🍵"

### 대화 선택지 라벨
`"오늘 많이 힘들었어요"` · `"짧게만 말해줘요"` · `"그냥 같이 있어줘요"` · `"회복 방법이 뭐예요?"`

### 안전 폴백 대사
> "오늘 어떤 하루였든, 세이브는 됐어요. 잘 왔어요. 🌿"

### P0 폴백 표현
원형 `#ff9bb5` + 흰 테두리 2px / 아이콘 ❤️(또는 포션) `#ff9bb5` / 라벨 "힐러".

### 금지(시각)
사실적 의료복(간호사 유니폼) · 차가운 자주/검정 대량 사용.

# MVP NPC 02 — 여관주인 🟤

> **한 줄 정체성:** 밤의 여관을 홀로 운영하는 주인장. "오늘 방은 비어 있어요"가 입버릇.
> **역할:** 환대·쉬어가기 (쉬움 모드 / 복귀). **상징 색:** `#a89060`(우드 브라운).

### 배경 서사
이 여관은 길드 공식 숙소가 아니다. 브리핑도, 보상 정산도 없는 곳 — 그냥 자고, 먹고, 쉬는 곳. 젊을 땐 상급 모험가였지만 어느 날 스스로 그만뒀다. 이유는 간단하다: **쉬는 법을 몰라 계속 부서졌고, 부서지는 게 지겨워졌다.** 지금은 밥을 짓고 이불을 개고 늦게 오는 모험가에게 문을 열어준다. 화려한 위로는 없다 — 대신 밥은 항상 따뜻하고 방은 항상 비어 있다. 여기서는 뭘 '해야' 하는 게 없다.

### 외형
- **실루엣/체형:** 직사각형의 묵직한 몸형(어깨 넓음), 중년의 성숙함. 선 굵기 힐러보다 두껍게(2.5~3px).
- **헤어:** 턱 아래 짧은 보브, 약간 웨이브. 베이스 `#a89060`(우드 브라운) + 하이라이트 `#d4a574`, 좌측 흰머리 한 줄 `#e8d5c4`.
- **의상:** 팔꿈치 소매 린넨 셔츠 `#e8d5c4` + 앞치마 `#b8956a`(주머니 2개 `#a89060`) / 면 바지 `#d4a574` / 앞치마 허리 매듭 리본 `#ffb6d9` / 목 스카프 `#e8d5c4`.
- **대표 소품:** 왼손에 따뜻한 찻잔(잔 `#c4a474` + 액체 `#b5704d`) 또는 오래된 청동 열쇠(`#8b7355` + 고리 `#a89060`).

### 컬러 팔레트
| 역할 | hex |
|---|---|
| 메인(셔츠) | `#e8d5c4` |
| 서브(앞치마) | `#b8956a` |
| 액센트(리본) | `#ffb6d9` |
| **상징 색**(틴트) | `#a89060` |

### 표정 변형 5종
1. **기본** — 중간 크기 눈(성숙), 동공 `#4a4a4a`+하이라이트 1개, 눈썹 직선. 따뜻한 대칭 미소 `#c4a474`.
2. **위로(걱정)** — 눈썹 안쪽 모임, 눈가 살짝 주름(2~3px), 입 일직선 `~` `#9d8b7e`.
3. **기쁨** — 눈 반달, 눈썹 밝은 호, 입 'U'/'0' `#d4a574`.
4. **놀람·격려** — 눈 크게(놀란 기쁨), 눈썹 올림, 입 'O'.
5. **나른(쉬어)** — 눈 반쯤 감김, 눈썹 'ㄷ'자, 입 작은 'ㅣ' 또는 미약한 미소 `#c4a474`.

### 포즈 변형
- **기본:** 양손 앞치마 높이로 모으거나 찻잔 들고 살짝 숙임(환대).
- **음료 대접:** 오른손 찻잔 내밀고 왼손 자연스럽게(베테랑 서빙).
- **응원:** 양팔 편안히 내리고 가슴팍 높이에서 살짝 앞으로(믿음·안정).

### 말투 규칙
- 호칭 "손님"(자주 오면 "이 친구"·"우리 단골"·이름).
- 툭툭 던지는 말투("-네요/-구먼/~하세요"), 과하게 공손 X.
- 담백·짧게. 음식·잠·몸 챙기는 실용 언어 자주.
- 이모지 🍲🛏️🕯️🌙 아주 가끔, 없어도 됨.
- **금지어:** "파이팅", "할 수 있어요!", "노력하면", "대단해요!", "열심히" (응원·과장·동기부여 전면 금지). **치어리더가 아니라 집주인.**
- 말보다 행동(밥·방·이불)으로 보살핌. 감정적으로 흥분 X.

### 대사 샘플
- **생존:** "일단 앉아요. 뭐 먹었어요 오늘? 🍲" / "말 안 해도 돼요. 방 있어요. 오늘은 그냥 자고 가요."
- **쉬움:** "잘 선택했네요. 쉬는 날 쉬는 게 제일 어려운 거예요."
- **보통:** "무난한 하루였나요. 그런 날이 사실 제일 좋은 날이에요."
- **도전:** "수고했네요. 지금 당장 뭔 말도 필요 없고 — 밥부터 먹어요. 🌙"
- **컴백:** "왔네요. 방 아직 있어요. 들어와요." / "얼마 만이든 상관없어요. 밥 줄게요 — 배고프죠?"
- **무기력 개입:** "요즘 몸이 말을 안 듣나요. 억지로 하려다 더 망가지는 거 나도 겪어봤어요. 오늘은 그냥 여기 있어요. 🍲"

### 대화 선택지 라벨
`"밥 주세요"` · `"그냥 있어도 돼요?"` · `"오늘 힘들었어요"` · `"짧게 한마디만"`

### 안전 폴백 대사
> "어쨌든 왔잖아요. 방 있어요. 들어와요."

### P0 폴백 표현
원형 `#a89060` + 흰 테두리 2px / 아이콘 🕯️(또는 찻잔) `#d4a574` / 라벨 "여관주인".

### 금지(시각)
너무 어둡거나 차가운 갈색(시니컬) · 거친 텍스처/과장식 앞치마.

# 확장 NPC (P2) — 스케치

> P2에서 일러스트·말투·대사 풀을 깊게 제작. 지금은 방향만.

### 길드마스터 🔵
- **정체성:** 길드 2층 집무실의 베테랑 마스터. 다음 한 걸음을 찾아주는 사람 — 등을 밀되 넘어뜨리지 않는. (역할: 코칭/보통 모드)
- **말투:** 든든·간결한 존댓말, 현실적 코칭형.
- **외형:** 넓은 어깨·당당한 체형, 작은 인장/퀘스트판. 헤어 `#4a5f8f`(네이비)+골드 액센트, 자켓 `#2c3e5f`, 골드 버튼 `#d4af37`. 상징 색 `#4a5f8f`. 폴백 아이콘 퀘스트판/인장 `#d4af37`.
- **대표 대사(보통):** "지금 당신한테 맞는 난이도, 한 단계 아래예요. 그게 약한 게 아니라 전략이에요." / "막혀 있으면 말해요. 길은 하나가 아니에요."

### 라이벌 🔷
- **정체성:** 같은 길드 소속의 까칠한 동기. 도발하지만 절대 혼자 두지 않는 사람. (역할: 동기부여/도전 모드)
- **말투:** 반말 섞인 짧고 직선적. 도발 뒤 반드시 챙기는 한 마디. ⚠️ 위기 신호 시 도발 중단→힐러 톤 전환.
- **외형:** 날렵한 체형, 검/깃발. 헤어 `#5f8fa3`(시안)+`#c74a5c` 스트라이프, 경량 전투복 `#5f8fa3`+빨강 라인 `#c74a5c`. 상징 색 `#c74a5c`. 폴백 아이콘 검 `#c74a5c`.
- **대표 대사(도전):** "뭐야, 그게 한계예요? — 거짓말. 아직 남아있잖아요." / "지금 포기하면 내가 더 앞서가는 거 알죠? 그래도 괜찮으면 쉬어요. 진짜로."

### 마법사 🟣
- **정체성:** 탑 꼭대기의 4차원 마법사. 선택지를 정리해주되 결론은 먼저 말하지 않는 사람. (역할: 명료화/결정장애 태그)
- **말투:** 신비롭고 약간 어긋나되 논점은 정확. "흥미롭군요" 자주.
- **외형:** 지팡이+떠다니는 별 오라. 헤어 `#b8a3d4`(라벤더)+`#e8d5f5`, 로브 `#b8a3d4`+골드 별 자수 `#d4af37`, 지팡이 끝 별 `#d4af37`. 상징 색 `#b8a3d4`. 폴백 아이콘 별/지팡이 `#d4af37`.
- **대표 대사(결정장애):** "선택지가 두 개처럼 보이죠. 근데 세 번째가 있어요 — 아직 아무것도 안 하는 것. 그것도 선택이에요." / "흥미롭군요. 당신이 고민하는 건 선택이 아니라 스스로한테의 허락인 것 같아요."

## 제작 트랙 메모 (team-lead)
- **P0:** 폴백(원형 실루엣+아이콘+라벨)만으로 루프 동작 → 일러스트 없어도 출시 가능.
- **일러스트 우선순위:** 힐러 → 여관주인 순(MVP 2종). 각 캐릭터당 *표정 5종 + 포즈 2~3종*이 1차 에셋 묶음.
- **확장 시:** NPC 3종 추가 = 일러스트 작업량 그만큼 증가(표정·포즈 변형 포함)를 일정에 반영.
- **다음 후보:** 힐러 1차 에셋(기본 포즈+표정 5종) 시안 기준표, 또는 대사 풀 확장(모드별 3~5개씩).

## 부록 A. 힐러 1차 에셋 기준표

> 일러스트레이터가 실제로 그릴 때 보고 따라가는 제작 명세. (designer 작성)

### A-1. 캔버스·사이즈 규격

| 용도 | 기준 사이즈 | @2x | @3x | 안전영역 | 포맷 |
|---|---|---|---|---|---|
| 대화창(상반신) | 360×480px | 720×960 | 1080×1440 | 상하좌우 16px | PNG(투명, sRGB) |
| NPC상담소 카드(반신) | 240×240px | 480×480 | 720×720 | 반경 20px | PNG(투명) |
| 아바타(원형) | 120×120px | 240×240 | 360×360 | 반경 16px | PNG(투명) |

대화창은 어깨~머리 상단까지 여유. 모바일 @2x/@3x 필수.

### A-2. 그리드·비율 (360px 캔버스 기준)
- 치비 비율: 머리 30 / 몸통 70 단위. 머리 중심선 y≈28, 눈높이 y≈20, 어깨선 y≈55.
- 얼굴 폭 80px / 눈 사이 28px / 눈 20×24px(세로로 큼) / 눈썹 폭 22px·두께 3px(눈 위 8px) / 입 18×4px(눈 아래 32px) / 하이라이트 큰 것 8×10px(우상)+작은 것 4×5px(우하).
- 외곽선 전부 2~3px, 앤티에일리어싱.

### A-3. 파츠별 컬러 스와치

| 부위 | hex | 다크배경(`#1a1620`) 보정 |
|---|---|---|
| 헤어 베이스 | `#f0d4d9` | 외곽 `#d4a8b0` 또는 이너글로우 `#fef0f5` |
| 헤어 하이라이트 | `#fef0f5` | 베이스와 2px 블렌드 |
| 튜닉(메인) | `#f5e6e8` | 외곽 `#d9bcc0` 1~1.5px |
| 스커트(서브) | `#ffe6f0` | 외곽 `#e8b8cc` 1~1.5px |
| 띠·소매밴드(액센트) | `#ffb6d9` | 배경 명도차 >30 |
| 포션 액체=상징색 | `#ff9bb5` | 배경 위 명도차 최대 |
| 입 | `#f0a8b5` | 기쁨 `#ffb6d9` / 위로·나른 `#e89aa5` |
| 눈동자 | `#4a3f4a` | 흰 하이라이트 `#ffffff`와 대비 |
| 피부 | `#f5e0e3` | 입과 구분되게 |

> 다크배경 위 핵심: 파스텔 외곽선 추가 + 명도 L≥65 유지(배경 L≈8과 차 55+).

### A-4. 표정 5종 델타 (기본 대비)

| 슬롯 | 눈 | 눈썹 | 입 | 하이라이트 | 사용 |
|---|---|---|---|---|---|
| `default` 기본 | 크게 열림 | 부드러운 호, 위로 | 옅은 미소 `~` `#f0a8b5` | 2개 | 보통 |
| `comfort` 위로 | 약간 축소, 한쪽 살짝 감김 | 안쪽 모임(ㄱ) | 뒤집은 U, 색 옅게 | 작은 것만 | 생존·무기력·HP1 |
| `joy` 기쁨 | 반달로 감음 | 밝은 호, 올림 | U 환한 웃음 `#ffb6d9` | 1개 선명 | 컴백·완료 축하 |
| `cheer` 놀람·격려 | 동그랗게 크게 | 위로 올림, 정면 | O 감탄 | 2개 최대 | 도전·all_max |
| `relax` 나른 | 반쯤 감김 | 가로 일직선 | 작은 ㅣ/미약한 미소 | 없거나 흐리게 | 쉬움 |

> 표정 슬롯명은 [[NPC-반응규칙]] §2와 동일. 제작 시 **기본을 베이스로 눈·눈썹·입만 교체**.

### A-5. 포즈 3종
- `default` 기본: 양손 배 앞 모음, 머리 15° 좌측 기울임(경청). 팔꿈치 약간 잘려도 OK.
- `healing` 치유 제시: 오른손 포션 가슴 높이 앞으로, 왼손 가슴. 표정 joy/cheer.
- `embrace` 포옹하듯: 양팔 45° 벌림, 고개 살짝 숙임. 양팔 캔버스 끝 8~12px 여유. 표정 joy/comfort.

### A-6. 레이어 구조 & 네이밍
- 베이스 몸(스커트·튜닉·팔·머리) 고정 + **표정(눈·눈썹·입) 5종 폴더 분리** → 표정만 swap.
- 네이밍: `healer_[pose]_[expr].png` (예: `healer_default_comfort.png`), 베이스 `healer_[pose]_base.png`, 해상도 `_1x/_2x/_3x`. PNG RGBA.

### A-7. 1차 완료 판정 (체크리스트)
- [ ] 포즈 3종 × 기본표정(중립) — 최소 3장
- [ ] 기본포즈 × 표정 5종 — 최소 5장
- [ ] 소품: 포션병 + 손수건/하트 배지
- [ ] 다크배경(`#1a1620`) 위 시뮬: 명도차 30+, 외곽선/글로우 적용, 파스텔 안 뭉개짐
- [ ] 파일 네이밍·투명 PNG·@1x/2x/3x
- [ ] 소형(120px)에서 표정 분간 가능
- [ ] 파스텔·귀여움 사수: 사실적 음영 X, 선 2~3px, 따뜻한 저채도

### A-8. 귀여움 사수 원칙 (피할 것 → 할 것)
- 샤프한 1px 검정선·각진 모서리 → 2~3px 부드러운 곡선
- 고채도 원색·네온 → 파스텔 저채도(채도 60~75%)·따뜻함
- 사실 인체(1:2)·길쭉한 팔다리 → 치비 1.2:1·짧고 둥근 팔다리
- 사실적 음영·모공·리얼 텍스처 → 부드러운 면, 미니멀 선, 사탕처럼 둥근 모양
- 레퍼런스: Cozy Grove / Animal Crossing / Spiritfarer 의 치비+부드러운 선+파스텔.

> **여관주인 에셋 기준표는 동일 포맷으로 후속 작성**(상징색 `#a89060`, 소품 찻잔·등불·열쇠, 선 굵기 2.5~3px·중년 체형).
