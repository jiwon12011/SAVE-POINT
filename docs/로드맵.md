## 2차 — 3개 전부 예정 (사용자 결정: 다 해야 함)

### ① 결정의 가마 개입 (차별화 한 끗) — **우선순위 1**
- 결정 손님이 조제에 **실시간 개입**: 불안의 그림자 = 가마 액체를 흔들거나 탁하게, 다 탄 재 = 색을 식힘 등. 플레이어는 빻기/섞기로 **진정시켜 가며** 조제.
- "환자가 곧 퍼즐 환경" → Potion Craft와 가르는 핵심.
- **데이터 슬롯 예약됨:** `data.js` `CRYSTALS[].interfere`(현재 `null`).
- 규모: 큼 / 임팩트: 최고.
- [ ] 미착수

### ② 섞기 + 상점
- **섞기:** 가마 위 원 그리며 젓기 제스처 → 색 농도·소용돌이(손맛 depth). 1차에서 의도적 제외(§8). perf: 소용돌이는 가마 내부 한정 소형 캔버스(≤256²).
- **상점:** 별사탕으로 씨앗·약초 구매(무대 `bg-star-candy-shop`). 별사탕 쓸 곳 = 모으는 동기.
- [ ] 미착수

### ③ 히든 레시피 + 재배/채집
- **히든:** 정답 외 조합 일부에 숨은 포션 + 발견 연출(마법사 놀람 thinking 표정). 도감 별도 칸. "정답 맞히기→발견하기"로 권태 차단.
- **재배/채집:** 약초 수급 — 재배(`bg-seed-drawer`), 채집(`bg-moon-garden`·`bg-natural-forage`).
- [ ] 미착수
